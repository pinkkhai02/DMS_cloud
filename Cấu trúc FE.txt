📝 Giải thích cấu trúc thư mục
🎯 Core Directories
src/components/: Tất cả React components
ui/: shadcn/ui components
common/: Shared components được dùng nhiều nơi
forms/: Form-specific components
src/pages/: Page-level components với routing
Mỗi page có thư mục riêng với components con
src/hooks/: Custom React hooks để tái sử dụng logic
src/services/: API calls, external integrations
api/: HTTP requests
storage/: Local/session storage utilities
🔧 Configuration & Utils
src/store/: State management (Redux Toolkit/Zustand)
src/utils/: Helper functions, constants, utilities
src/types/: TypeScript type definitions
src/lib/: External library configurations
🎨 Styling & Assets
src/styles/: CSS files và Tailwind customizations
src/assets/: Images, icons, fonts
